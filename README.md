# 💗 Dönen Kalp Animasyonu

`love you` yazılarından oluşan, 3D dönen bir kalp animasyonu. Mesajını
kişiselleştirip arkadaşlarına link olarak gönderebilirsin.

## Canlı demo
GitHub Pages açıldıktan sonra link burada olacak:
`https://<kullanıcı-adın>.github.io/<repo-adı>/`

## Kişiselleştirme
Linke parametre ekleyerek mesajı değiştirebilirsin:

```
?word=love%20you&msg=Seni%20Seviyorum&color=ea80b0
```

| Parametre | Açıklama              | Örnek            |
|-----------|----------------------|------------------|
| `word`    | Kalbi oluşturan kelime | `love you`       |
| `msg`     | Alttaki mesaj         | `Seni Seviyorum` |
| `color`   | Renk (hex, # olmadan) | `ea80b0`         |

Sitedeki **✏️ Kendin oluştur** butonundan da düzenleyip **🔗 Linki kopyala**
ile hazır linki alabilirsin.

## Dosyalar
- `index.html` — sayfa
- `style.css` — tasarım & animasyonlar
- `script.js` — kalbi oluşturan kod + kişiselleştirme
